- [ ] 下拉菜单列出所有可用档案

### 4.4 时间轴控制（Timeline Sync）

## 五、端到端演示流程

按以下顺序模拟完整演示：

1. [ ] 打开首页 → 等待加载 → 展示晨报
2. [ ] 进入 Data Center → 切换 Tab → 展示图表
3. [ ] 点击 AI 解读 → 展示分析结果
4. [ ] 打开 AI Advisor → 发送问题 → 展示回复
5. [ ] 激活 God-Mode → 切换档案 → 展示数据变化
6. [ ] 注入事件 → 展示 Banner 和 Advisor 响应
7. [ ] 使用 Timeline Control 追加"进餐"片段 → 同步 → 查看数据变化
8. [ ] 追加"有氧"片段 → 推进时钟 1 小时 → 再次同步
9. [ ] 重置时间轴 → 确认恢复到初始晨间状态
10. [ ] 重置全部 → 确认恢复
