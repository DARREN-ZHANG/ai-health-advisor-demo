# Wave 1 — UI 包实施计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 构建可复用的 UI 组件库，包含设计 token、布局组件、状态组件、交互组件和反馈组件。

**Architecture:** 纯 React 组件库，使用 Tailwind CSS + CSS custom properties。暗黑主题为默认。所有组件接受 className prop 支持外部覆盖。

**Tech Stack:** React 19, TypeScript, Tailwind CSS 4, Vitest + @testing-library/react

**前置依赖:** Plan 1 (Shared 包) 完成后开始

```tsx
import { forwardRef, type ReactNode, type ButtonHTMLAttributes } from 'react';

interface ButtonProps extends ButtonHTMLAttributes<HTMLButtonElement> {
  variant?: 'primary' | 'secondary' | 'ghost';
  children: ReactNode;
}

const VARIANT_STYLES = {
  primary: 'bg-blue-600 text-white hover:bg-blue-700',
  secondary: 'bg-slate-700 text-slate-200 hover:bg-slate-600',
  ghost: 'text-slate-400 hover:bg-slate-800 hover:text-slate-200',
};

export const Button = forwardRef<HTMLButtonElement, ButtonProps>(
  ({ variant = 'primary', className = '', children, ...props }, ref) => (
    <button
      ref={ref}
      className={`inline-flex items-center justify-center rounded-md px-4 py-2 text-sm font-medium transition-colors disabled:opacity-50 ${VARIANT_STYLES[variant]} ${className}`}
      {...props}
    >
      {children}
    </button>
  ),
);
```

```tsx
import { forwardRef } from 'react';

interface TabItem {
  id: string;
  label: string;
}

interface TabsProps {
  items: TabItem[];
  activeId: string;
  onSelect: (id: string) => void;
  className?: string;
}

export const Tabs = forwardRef<HTMLDivElement, TabsProps>(
  ({ items, activeId, onSelect, className = '' }, ref) => (
    <div ref={ref} className={`flex gap-1 ${className}`}>
      {items.map((item) => (
        <button
          key={item.id}
          onClick={() => onSelect(item.id)}
          className={`rounded-md px-3 py-1.5 text-sm font-medium transition-colors ${
            item.id === activeId
              ? 'bg-slate-700 text-white'
              : 'text-slate-400 hover:bg-slate-800 hover:text-slate-200'
          }`}
        >
          {item.label}
        </button>
      ))}
    </div>
  ),
);

Tabs.displayName = 'Tabs';
```
