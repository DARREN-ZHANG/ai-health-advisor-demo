### 数据逻辑描述

- **A 巅峰表现型**：极强的副交感神经控制力。HRV 维持在高位，运动后恢复极快，生理节律如同教科书般稳定。标准体重、有规律健身、长睡眠(>7h)、无饮酒。
- **B 低效睡眠型**：虽然睡眠时长够，但受限于体重，存在睡眠呼吸暂停(OSA)风险。白天因夜间缺氧导致持续性疲劳。偏胖、无健身习惯、长睡眠(>7h)、无饮酒。
- **C 焦虑耗竭型**：长期处于慢性压力下，皮质醇水平高。皮温(NTC)常因末梢血管收缩而偏低，心率受情绪波动影响剧烈。偏瘦、无健身习惯、短睡眠(<6h)、无饮酒。
- **D 社交运动型**：身体机能好但生活方式冲突。运动带来的增益常被酒精代谢抵消，导致"恢复赤字"。标准/偏瘦、有氧运动偏多、中等睡眠(6-7h)、有饮酒。

## 新增 Segment Type

在现有 6 种 ActivitySegmentType 基础上新增 6 种：

| Type | 显示名 | 图标 | 默认时长 | 默认参数 | 适用场景 |
|------|--------|------|----------|----------|----------|
| `deep_focus` | 专注 | 🧠 | 120min | `{ intensity: 'high' }` | 深度工作、创意脑暴 |
| `anxiety_episode` | 焦虑 | 😰 | 30min | `{ trigger: 'work' }` | 焦虑发作、心慌、社交恐惧 |
| `breathing_pause` | 呼吸暂停 | 🫁 | 5min | `{ severity: 'moderate' }` | 睡眠呼吸暂停、憋醒 |
| `alcohol_intake` | 饮酒 | 🍺 | 60min | `{ amount: 'moderate' }` | 社交饮酒、持续饮酒 |
| `nightmare` | 噩梦 | 👻 | 10min | `{ intensity: 'high' }` | 噩梦惊醒 |
| `relaxation` | 放松 | 📖 | 60min | `{ activity: 'reading' }` | 阅读、冥想、晚间放松 |

## 生成器函数数据范围

### `generateDeepFocusEvents`
- 心率: 55-62 bpm
- HRV: 75-85 ms
- 运动强度: 极低（几乎零位移）
- SpO2: 99-100%
- 步数: 0

### `generateAnxietyEpisodeEvents`
- 心率: 85-110 bpm
- HRV: 8-18 ms
- 运动强度: 极低到中等（僵坐或高频微动）
- SpO2: 97-98%
- 步数: 0-5

### `generateBreathingPauseEvents`
- 心率: 88-95 bpm
- HRV: 8-10 ms
- 运动强度: 突发大幅动作
- SpO2: 86-94%（低氧区间）
- 步数: 0

### `generateAlcoholIntakeEvents`
- 心率: 85-98 bpm
- HRV: 8-15 ms
- 运动强度: 不规则（手部活跃/不规则步态）
- SpO2: 95-98%
- 步数: 0-20

### `generateNightmareEvents`
- 心率: 85-95 bpm
- HRV: 10-15 ms
- 运动强度: 突发大幅位移
- SpO2: 96-97%
- 步数: 0

### `generateRelaxationEvents`
- 心率: 50-55 bpm
- HRV: 85-92 ms
- 运动强度: 零位移
- SpO2: 99-100%
- 步数: 0
