## Data Detail Analysis

Generate a data analysis summary based on the health metric tab selected by the user.

### Output Requirements

- In-depth analysis of the currently selected metric
- Related chart metric token list
- Improvement recommendations for this metric

### Metric Descriptions

- HRV: Heart rate variability, reflects autonomic nervous system regulation capacity
- Sleep: Includes sleep duration, stage proportions, and sleep score
- Resting Heart Rate: Reflects baseline cardiopulmonary function
- Activity: Steps, calorie expenditure, active duration
- Blood Oxygen: Blood oxygen saturation level
- Stress: Comprehensive stress load score
